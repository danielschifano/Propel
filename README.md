#Propel
A prototype for a peer to peer application concept.

#Docs
Make sure that your development environment has Node.js installed.

#Features
* Basscss
* Gulp
* BrowserSync
* Minify
* Uglify
* Autoprefixer

#Installation
Run this on command line.
```
npm install
```
And then run following command.
```
gulp build
```
Last run and happy coding.
```
gulp
```
